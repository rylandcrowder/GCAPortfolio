## Commands

### Deploy
When I say "Deploy":

1. **Verify Location**
   - Confirm we're inside the GCAPortfolioSite folder (or a subfolder of it)
   - Check that .git exists at the root level

2. **Stage and Commit**
   - `git add .`
   - `git commit -am "Update: [describe what changed]"`

3. **Push**
   - `git push origin develop`

### Run Locally
Open `[github url]` in a browser to view the site.

## Coding Standards
- Keep HTML, CSS, and JS in single files when possible
- Use semantic HTML elements
- Mobile-responsive design

## File Naming
- Main file: `index.html`
